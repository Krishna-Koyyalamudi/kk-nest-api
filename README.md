# kk-nest-api

## Links
- [Source code](https://github.com/Krishna-Koyyalamudi/kk-nest-api)
- [App on Heroku](https://kk-nest-api.herokuapp.com/)

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
